# 巴菲特投資哲學

## 核心理念：心態
- 多數人只學技術，忽略心態
- 巴菲特買的是公司，不是股票
    - 我們在玩股票，他在買公司
- 心態建立：早年教訓與轉變
    - 早期：老師班傑明格拉漢「檢便宜」（煙屁股）
        - 買下波克夏，生涯最大錯誤
    - 轉變：查理蒙格影響
        - 找被低估的好公司
        - 棒球理論：等待甜蜜點，打全壘打
    - 反人性：
        - 市場恐慌時買入偉大企業
        - 市場瘋狂時不動如山
        - AI狂潮下：只持蘋果、亞馬遜兩檔相關股
- 價值投資的秘密
    - 反人性的心態
    - 忘掉股價波動，專注企業價值
    - 把自己當成公司擁有者

## 掌握巴菲特投資精髓的四步驟
- 工具其次，理解背後道理
- 哲學簡單，考驗紀律與耐心
- 四個主要步驟：
    - 1. 建立能力圈：不懂不投，從興趣產業學起
    - 2. 看懂財報：
        - 產品能否長期漲價？管理層是否可靠？
        - 關注ROE（股東權益報酬率），需搭配其他數據
    - 3. 找到護城河：
        - 公司獨特優勢
        - 測驗：產品漲價10%顧客還會買嗎？（例：可樂、蘋果）
    - 4. 讓雪球滾起來：
        - 不打算持十年就別買
        - 長期投資，複利發揮威力
        - 不想研究個股，ETF也是好選擇

## 總結
- 價值投資：買打折的好公司，不是便宜貨
- 能力圈：市場瘋狂也只在能力圈內揮棒
- 投資真諦：找到好生意，讓時間成為朋友
