# SnowPro® Advanced: Data Analyst

- The SnowPro® Advanced: Data Analyst Certification Exam will validate advanced knowledge and skills used to apply comprehensive data analysis principles using Snowflake and its components.
- https://learn.snowflake.com/en/certifications/snowpro-advanced-dataanalyst

## About Certification

**SNOWPRO ADVANCED: DATA ANALYST OVERVIEW**

This certification will test the ability to:

- Prepare and load data
- Perform simple data transformations for data analysis
- Build and troubleshoot advanced SQL queries in Snowflake
- Use Snowflake built-in functions and create User-Defined Functions (UDFs)
- Perform descriptive and diagnostic data analyses
- Perform predictive data analysis
- Prepare and present data to meet business requirements

**SNOWPRO ADVANCED: DATA ANALYST CANDIDATE**

1 or more years of hands-on practical experience with Snowflake as a Data Analyst in a production environment. 

In addition, successful candidates may have fluency with advanced SQL and knowledge of an additional computer language.
