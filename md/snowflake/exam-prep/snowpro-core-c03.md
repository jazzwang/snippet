# SnowPro® Core Certification (COF-C03)

- The SnowPro® Core Certification will validate practical, hands-on experience with the Snowflake AI Data Cloud.
- https://learn.snowflake.com/en/certifications/snowpro-core-c03

> *This exam version is replacing SnowPro Core (COF-C02) and launched on February 16, 2026. For more information review these [FAQs](https://publish-p93462-e887935.adobeaemcloud.com/content/dam/SnowPro-Core-FAQS.pdf).*

**SNOWPRO CORE CERTIFICATION OVERVIEW**

This certification will test the ability to:

- Use the Snowflake AI Data Cloud architecture
- Manage Snowflake accounts and virtual warehouses
- Perform data loading, unloading, and transformation
- Use structured, semi-structured, and unstructured data
- Monitor and optimize performance
- Enable data collaboration and protection
- Establish Snowflake connectivity

**SNOWPRO CORE CANDIDATE**

6 or more months of experience using Snowflake
