# Rufus

> Create bootable USB drives the easy way

- Git Repo
  - https://github.com/pbatard/rufus
- Website
  - https://rufus.ie/en/

## 2025-11-02

- 根據 Clonezilla 文件推薦，在 Windows 上可以用 Rufus 來製作 Clonezilla Live
  - https://clonezilla.org/liveusb.php#windows-method-b
- 安裝：
  - 用 `scoop` 安裝
  ```bash
  ~$ scoop install rufus
  ```
- 目標：製作 GParted Live USB
  - 下載：https://sourceforge.net/projects/gparted/files/gparted-live-stable/
