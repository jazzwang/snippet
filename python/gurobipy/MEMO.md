# Gurobi Optimizer

> Gurobi Optimizer 是一個數學最佳化軟體庫，用於解決混合整數線性和二次最佳化問題。
> The Gurobi Optimizer is a mathematical optimization software library for solving mixed-integer linear and quadratic optimization problems.
