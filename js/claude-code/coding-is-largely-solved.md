影片連結：[Head of Claude Code: What happens after coding is solved | Boris Cherny](https://www.youtube.com/watch?v=We7BZVKbCVw)

這段影片是 **Lenny's Podcast** 對 Anthropic 的 Claude Code 負責人 **Boris Cherny** 的專訪。影片深入探討了 AI 如何徹底改變軟體開發流程，以及在「寫程式問題已被解決」的未來，工程師與產品經理的角色將如何演變。

以下是影片的核心內容摘要：

### 1. Claude Code 的崛起與影響力
* **寫程式已不再是瓶頸：** Boris 提到他現在 100% 的程式碼都是由 Claude Code 編寫的，他自 2025 年 11 月以來就沒有手寫過任何一行程式碼 [[00:00](http://www.youtube.com/watch?v=We7BZVKbCVw&t=0)]。
* **極高的生產力：** 即使身為團隊負責人，Boris 每天仍能透過 AI 代理（Agents）交付 10 到 30 個 PR（Pull Requests），工程師的生產力提升了 200% [[00:06](http://www.youtube.com/watch?v=We7BZVKbCVw&t=6)]。
* **爆發式成長：** Claude Code 在發布後，日活躍用戶（DAU）呈現垂直式成長，並成為 Anthropic 估值飆升至 3500 億美元的重要推動力之一 [[01:21](http://www.youtube.com/watch?v=We7BZVKbCVw&t=81)]。

### 2. 軟體工程的未來轉型
* **從「工程師」到「建造者」：** Boris 預測「軟體工程師」這個職稱將逐漸消失，取而代之的是「建造者」（Builder）。未來寫程式將變得無處不在，任何人都能隨時構建軟體 [[00:48](http://www.youtube.com/watch?v=We7BZVKbCVw&t=48)]。
* **Coding 已被「解決」：** 他大膽預言，在未來一兩年內，學習單純的寫程式技術將不再重要，因為這項任務基本上已被 AI 解決 [[00:21](http://www.youtube.com/watch?v=We7BZVKbCVw&t=21)]。
* **人人都是產品經理：** 隨著技術門檻降低，未來每個人都會更專注於「產品思考」，每個人都將具備產品經理的職能 [[00:44](http://www.youtube.com/watch?v=We7BZVKbCVw&t=44)]。

### 3. AI 代理的工作模式
* **像同事般的 AI：** Claude 開始能夠主動提出想法、分析使用者回饋、查看錯誤報告並處理遙測數據來修復 Bug，其表現更像是一個「同事」而非單純的工具 [[00:30](http://www.youtube.com/watch?v=We7BZVKbCVw&t=30)]。
* **快速修復 Bug：** 透過 Twitter 等社群平台收集到的回饋，Boris 可以在幾分鐘內利用 Claude 修復 Bug 並部署，這種速度在過去是難以想像的 [[01:26:15](http://www.youtube.com/watch?v=We7BZVKbCVw&t=5175)]。

### 4. Boris 的個人背景與職涯抉擇
* **閃電離職與回歸：** Boris 曾短暫離開 Anthropic 加入 Cursor，但兩週後便決定回歸。主要原因是對 Anthropic 「安全性第一」使命的強烈認同感 [[04:17](http://www.youtube.com/watch?v=We7BZVKbCVw&t=257)]。
* **熱愛開發：** 他表示自己從未像現在這樣享受寫程式，因為 AI 幫他處理了所有瑣碎的細節，讓他能專注於更高層次的創造 [[00:15](http://www.youtube.com/watch?v=We7BZVKbCVw&t=15)]。

### 5. 給開發者的建議
* **培養「常識」：** 在 AI 時代，最重要的技能是開發自己的「常識」（Common Sense）與判斷力，去判斷 AI 生成的方案是否合理 [[01:24:35](http://www.youtube.com/watch?v=We7BZVKbCVw&t=5075)]。
* **擁抱實驗：** 創新沒有路線圖，需要給予團隊失敗的空間（心理安全感），並不斷嘗試推動模型極限 [[01:15:23](http://www.youtube.com/watch?v=We7BZVKbCVw&t=4523)]。

這場訪談揭示了一個新時代的到來：寫程式技術將商品化，而**解決問題的創意**與**產品定義能力**將成為核心競爭力。
