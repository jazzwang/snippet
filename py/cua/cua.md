# cua

> Open-source infrastructure for Computer-Use Agents. Sandboxes, SDKs, and benchmarks to train and evaluate AI agents that can control full desktops (macOS, Linux, Windows).

## 2026-05-05

- 緣起：在 Linkedin 看到 https://www.linkedin.com/feed/update/urn:li:share:7453816386222465024/

> 最近 Computer-Use Agent 愈來愈紅，但有一個共同痛點：要讓 Agent 真的能「看螢幕、點按鈕、敲鍵盤」，你得先準備一台沙盒環境，最好還能跨作業系統。trycua 團隊推出的 Cua 就是要解決這問題。
>
> 最大賣點是同一套 Python API 操控任何 OS，pip install cua 裝完就能截圖、點滑鼠、打鍵盤、做多點觸控手勢。
